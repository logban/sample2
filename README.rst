Requirement
===========
 
:Python: 3.6
:Django: 1.11
 
 
Quick start
===========
1. プロジェクトのクローン::
 
    git clone https://github.com/logban/django_sample2.git
 
2. モデルのmigrate::
 
    docker-compose run web python manage.py migrate
 
3. うごかす::
 
    docker-compose run web python manage.py runserver
